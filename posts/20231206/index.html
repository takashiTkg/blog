2023-12-06

こんばんは、たかしです。
自分の考えてることを整理するために日記を書く習慣をつけようと思います。

今日やったこと#

  • なんか元気が出なかったので寝てた
  • やらないといけない事を頭の中で考えてグルグルしていた
  • やる気が出ないときに意識的に対処する必要があると考えた

頭の中を整理するために#

思考がどこかで詰まってすっきりしない場合、どうにかして整理しないと非常に気持ちが悪い。
一般的には書き出したりして整理すると思うので、僕もそれに倣おうと思う。

やりたいこと#

Misskey のノートをリアクションで検索するやつ#

  • pagination の実装
  • チャンネルへのリアクションを除外する処理をサーバーサイドでやる
  • daisyUI のテーマをいい感じのやつにして配色をなんとかする

アドカレ#

  • SOFT SKILLS を読んだ感想というかまとめを書きたいので SOFT SKILLS を読む
  • hugo new して予約投稿しておく

TUI Misskey クライアント#

  • Go で適当に書いたら破滅したので書き直す
  • モジュールをちゃんと分けて書く

ブログ#

  • 全体的にバージョンが古いので更新したい
  • 特に Node と tailwind
  • Tailwind Typography のスタイルを調整してマージンをもう少し小さくしたい(たぶんバージョン上げないとできない)

学習#

  • リーダブルコードを読む
  • ポートフォリオが中途半端になっているので続きを作る

やらないといけないこと#

お部屋探し#

  • すでに何件かいい感じのところは探せたので問い合わせをする

片付け&荷造り#

  • 掃除とか
  • 服の整理
  • 年末までにいらないものを捨てる
  • 必要なものを揃える

引っ越しに必要な手続き等#

  • やらないといけない手続きを書き出す
  • 転出届とか転入届とか

最後に#

書き出したらちょっとすっきりした。
考えていることを吐き出さないと次のことを考えられない。
なんて事は少し考えればわかることなのに、なかなか出来ないのが悩み。
習慣にして、毎日書くように頑張りたい。
